General usage

If you want to translate a local extension, this is already selected by default. Just select the extension where you want to translate labels.

Select there xlf file where the labels are stored that you want to translate.

Select the language you want to translate. If the language you want to translate does not exist in the list, either the language does not exist the system or your administrator does not allow this language to translate.
